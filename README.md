# YT Translate

YouTube 沉浸式双语翻译浏览器扩展。

## 功能

- **字幕翻译** — 实时双语字幕覆盖层，基于 `requestAnimationFrame` 与视频同步
- **评论翻译** — 评论区双语对照翻译
- **标题 & 简介翻译** — 视频标题和简介内联翻译
- **直播弹幕翻译** — 实时聊天消息翻译

## 技术栈

- Chrome Extension Manifest V3
- DeepSeek v4 Flash API
- Vanilla JavaScript

## 安装

1. 打开 Chrome/Edge，进入 `chrome://extensions`
2. 启用「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本目录
4. 打开 YouTube 即可使用

## 配置

点击扩展图标 → 在弹窗中配置 API Key 和功能开关。
