# 隐私政策 (Privacy Policy)

## YT Translate 浏览器扩展

**最后更新日期：2026-07-20**

### 数据收集与使用

YT Translate 扩展**不收集、不存储、不上传**任何用户个人信息。

- **API Key**：用户自行配置的 DeepSeek API Key 仅存储在浏览器本地（chrome.storage.local），不会发送到除 DeepSeek API 之外的任何服务器。
- **翻译内容**：YouTube 页面上的文本内容（字幕、评论等）会发送到 DeepSeek API 进行翻译处理，本扩展不保存或记录这些内容。
- **浏览数据**：本扩展不追踪、不收集用户的浏览历史或任何行为数据。

### 第三方服务

本扩展使用 DeepSeek API（https://api.deepseek.com）进行翻译。翻译请求受 DeepSeek 服务条款和隐私政策约束。
